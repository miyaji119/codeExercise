#### 基本功能

  + 展示文件夹列表
    - http://localhost
  + 显示文件夹中的图片文件列表
    - http://localhost/noct
  + 新增文件夹
    - http://localhost/new
  + 修改图片文件夹名称
    - http://localhost/noct/modify
  + 删除图片文件夹中的名为IMG_2343的图片
    - http://localhost/noct/IMG_2343/remove
  + 上传图片到某个文件夹
    - http://localhost/noct/upload
